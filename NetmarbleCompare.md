# NetmarbleCompare 전용 규칙

이 파일은 `NetmarbleCompare` 폴더 내에서 작업할 때만 추가로 적용되는 Claude 규칙입니다.
최상위 `CLAUDE.md`의 기본 규칙과 함께 병합되어 적용됩니다.



# ExcelMerge 주요 기능

- 파일 로딩
    - 드래그 앤 드롭 — A/B 패널에 엑셀, json, uasset 파일을 끌어다 놓아 로드
    - 찾아보기 버튼 — 파일 탐색기에서 파일 선택 (아이콘 버튼)
    - 커맨드라인 인자 — P4V 등 외부 툴에서 -s <파일A> -d <파일B> 형식으로 자동 실행

- 비교(Diff)
    - 자동 비교 — A/B 파일이 모두 지정되면 즉시 비교 실행
    - 키 열 기반 비교 — 지정한 열 값을 키로 행 매칭 (기본: A열)
    - ROW 순서 비교 — 키 열 해제 시 행 번호 순서대로 1:1 매칭
    - 색상 구분 표시
        🟢 연두: 한쪽 파일에만 값이 있는 셀 (A 전용/B 전용 모두 '추가됨')
        🟡 노랑: 양쪽 값이 서로 다른 셀 (변경됨)
        🟠 주황: 병합 준비 중인 셀
        🔵 연파랑: 병합 완료된 셀 — 저장 후에도 '변경 행만 보기'에서 계속 표시 (새 비교/새로고침 전까지)
    - 변경 행만 보기 — 변경된 행만 필터링 표시 (Ctrl+D)
    - 새로고침 — 지정된 경로의 파일을 다시 불러와 재비교

- 병합(Merge)
    - 셀 단위 병합 준비 — 셀 선택 후 우클릭 → A→B / B→A 방향 지정
    - 행 단위 병합 준비 — 행 헤더 우클릭으로 행 전체 방향 지정
    - 열 단위 병합 준비 — 열 헤더 우클릭으로 열 전체 방향 지정
    - 병합 준비 취소 — 우클릭으로 staged 셀 선택 해제
    - 수식 보존 병합 — 병합 시 계산값이 아닌 원본 수식(=...)으로 파일에 기록

- 저장
    - 파일별 개별 저장 — A 파일 저장 / B 파일 저장 버튼 (각 패널 내 아이콘 버튼)
    - 파일 잠금 감지 — 대상 파일이 Excel에서 열려 있으면 저장 불가 경고
    - 행 번호 자동 재정렬 — 행 삭제 후 행 번호 갭 없이 재정렬

- 셀 값 표시란 (v133~ 직접 편집 기능 제거됨 — 읽기전용)
    - 셀 선택 시 하단 표시란에 해당 셀의 값(수식 셀은 원본 수식)을 표시
    - Ctrl+Z 실행 취소 — 병합 준비 동작 단계별로 되돌리기
    
- UI/조작 편의
    - 양쪽 스크롤 동기화 — 가로/세로 스크롤을 A/B 패널이 연동
    - 선택 셀 동기화 — 한쪽 패널에서 셀 선택 시 반대쪽 패널도 동일 셀 선택
    - 키 열 헤더 표시 — 현재 키 열에 🔑 아이콘 + 노란 배경 표시
    - 상태바 안내 — 비교 결과(행수, 열수, 변경 셀 수) 및 현재 작업 상태 표시

- 변경된 행만 보기 토글
    - 비교 실행 시 디폴트로 변경된 행만 보기
    - 토글 시 변경된 행만 보거나 전체 행을 볼 수 있음

- 이전/다음 변경점으로 이동
    - 변경된 행만 보기에서 변경된 행으로 이동

- 찾기 (v124~)
    - 찾을 내용 검색란 — "다음 변경 ▶" 버튼 우측에 상시 노출, Ctrl+F로 포커스
    - Enter = 다음 찾기, Shift+Enter = 이전 찾기, ◀/▶ 버튼으로도 이동
    - Ignore case 토글 (Aa 아이콘, 기본 켜짐) — 대소문자 무시 여부, 툴팁 도움말
    - Match whole word only 토글 (ab 아이콘, 기본 꺼짐) — 전체 단어 일치 여부, 툴팁 도움말
    - 검색 범위: 화면에 보이는 행만 (변경된 행만 보기 시 숨겨진 행 제외), A/B 표시 텍스트 중 한쪽이라도 일치하면 매치
    - 끝에 도달하면 반대편에서 순환하며 상태바에 "n/전체 일치" 표시
    - 미리보기 검색 (v133~) — 파일이 하나만 로드된 미리보기 상태에서도 검색 가능



# 데이터 표시 규칙

 - 셀 값 표시란 (읽기전용)
    - A파일과 B파일에서 가져온 값을 그대로 표시.

- 비교 시트란
    - A파일과 B파일에서 가져온 값을 엑셀처럼 계산해서 표시.
